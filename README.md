
**<h1>Sistema Bancario Escalable - Patrones de Diseño 🏦</h1>**

<h3> Este proyecto consiste en una aplicación de software desarrollada en Python que simula un sistema bancario modular. Se centra en la aplicación de patrones de diseño de alto nivel para garantizar soluciones informáticas escalables y robustas.</h3>

**<h1> 🚀 Resultado de Aprendizaje </h1>**
<h3> El objetivo principal es demostrar la capacidad de desarrollar aplicaciones utilizando patrones de arquitectura que permitan la flexibilidad y el mantenimiento eficiente del software en entornos competitivos.</h3> 

**<h1>🛠️ Patrones de Diseño Implementados</h1>**
<h3>Para cumplir con los requerimientos de la actividad práctica, se han aplicado los siguientes patrones:</h3>

- **<h2>Singleton:</h2>** <h3>Se utiliza para la gestión del núcleo del banco **(BancoCentral)**, asegurando que solo  exista una instancia controlando el flujo de transacciones y garantizando la integridad de los datos.</h3>

- **<h2>Factory Method: </h2>** <h3>Se emplea para la creación de diferentes tipos de cuentas **(Ahorros, Corriente)**, permitiendo que el sistema sea escalable al añadir nuevos productos sin modificar el código base.</h3>

**<h1>📁 Estructura del Proyecto </h1>**
<h3> La aplicación sigue una organización profesional de paquetes para asegurar que el código esté bien estructurado y sea accesible:</h3>

* 📂 **SISTEMA\_BANCARIO\_PYTHON/**
    * 📂 **src/** — *Lógica del negocio y patrones*
        * 📄 `__init__.py`
        * 📄 `banco_singleton.py`
        * 📄 `fabrica_cuentas.py`
        * 📄 `modelos.py`
    * 📄 `main.py` — *Punto de entrada*
    * 📄 `README.md`
